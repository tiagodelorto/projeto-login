# Projeto - Tela de Login

#### Descrição

[Tiago Delorto](https://github.com/tiagodelorto) Introduzi os conceitos de framework, apresentando o Bootstrap e alguns exemplos disponíveis nesse framework, e git, realizando o passo a passo de configuração do ambiente de desenvolvimento

### Ferramentas Utilizadas

- HTML5
- CSS3
- Bootstrap
- Git

### Como visualizar

- Faça o download ou clone este repositório.

- Abra o arquivo index.html em qualquer navegador (Chrome, Edge, Firefox).

### Principais aprendizados

- Grid System: Usei o Bootstrap para centralizar o formulário no meio da tela (famoso "center a div").

- Componentes: Implementação de campos de input, checkbox de "Lembrar Senha" e botões estilizados.

- Git: Aprendi a inicializar um repositório, fazer commits e subir o código para o GitHub.
